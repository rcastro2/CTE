  1. Overview

    Career & Technical Education (CTE) in Computer Programming at Bayside High School provides a rigorous academic course of study which prepares students for post-secondary education and careers in computer programming.
    •   Students develop a strong foundation in programming concepts and gain practical experience programming in several languages.
    •   Students acquire the confidence to use their existing knowledge and apply it to learning advance programming concepts, new programming languages or new technologies and techniques.
    •   Students earn college credits and are motivated to continue their education in computer programming.
    •   Students observe first hand careers in computer programming and participate in work based experiences.

  2. Programming

    Throughout all four years emphasis is placed on developing computational thinking skills and learning programming concepts.
    •   Students are exposed to problems which incorporate various disciplines such as mathematics, physics, business, language, music and art.
    •   Students develop stand-alone applications, process data from external sources, design web sites through code, create basic games, generate code to control Lego MindstStorm, and produce mobile apps.
    •   Students explore programming concepts using a myriad of different languages and technologies such as Python, HTML, CSS, Bootstrap, JavaScript / JQuery, variuos APIs, C#, Unity 3D, Java, Andriod SDK, and various web and data store technologies.
    GOAL: Recognize that programming is everywhere. Algorithm is key!

  3. Academics
